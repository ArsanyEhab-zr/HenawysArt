# Henawy's Art - E-commerce Website

A modern, responsive e-commerce website for handmade paintings on natural wood slices, custom medals, and frames.

## Tech Stack

- **React.js** (Vite)
- **Tailwind CSS**
- **Framer Motion** (animations)
- **Lucide React** (icons)

## Features

- Responsive design
- Smooth animations with Framer Motion
- Custom color palette (Turquoise/Teal & Gold)
- Watercolor-inspired design elements
- Product catalog with pricing
- Shopping cart functionality (coming soon)

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Color Palette

- **Primary (Turquoise):** #66C2BD
- **Accent (Gold):** #ECC846
- **Text:** #2C2C2C
- **Background:** #FFFFFF

## Typography

- **Headings:** Dancing Script (handwritten/script font)
- **Body:** Cairo (clean sans-serif)
